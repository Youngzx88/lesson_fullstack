- js 的面向对象
    1. Js是一种基于对象的语言
    2. JS又不是一种真正的面向对象(OOP)语言：关键字早期不提供class

- 生成实例对象的原始模式
    js 内置的对象字面量 
    1. 繁琐->
    2. 没有什么约束
- 原型模式的改进
    1. 构造函数，封装实例化赋值属性的过程，首字母大写+ this + new (js语言的设计)
    2. js是面向对象的语言，但是不同于普通的OOP，一切皆对象，面向对象更纯纳粹
    3. 为了简化实例化过程，使用函数封装，this指向等下会返回的实例，n       this指向实例．
