### npm: node package management

#### react创建:
- npm init @vitejs/app初始化脚手架
- vitejs当前最快的脚手架
    1. 输入项目名称
    2. 确定用react
    3. 得到了一个项目模板
#### 下一步
- cd 项目目录
- npm i根据项目描述，把项目依赖(dependencies+devDependencies)安装
- npm run dev
#### 介绍项目模板
- 项目模板

        - 挂载点 #root
        - script引入 src/main.jsx

            2. main.jsx入口文件
